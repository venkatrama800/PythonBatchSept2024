## Git Commands

### To clone a repository,
    git clone https URL

### To stage the changes
    git add <filename>

### To commit the changes
    git commit -m "commit message"

### To push the changes
    git push origin <source branch>

    class01 -> main
    git push origin class01

### To check/ verify the modified content in existing file
    git diff <filename>
